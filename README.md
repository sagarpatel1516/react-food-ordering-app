# React Food Ordering App

A modern food ordering application built with React and Redux Toolkit.

## Features

- Browse menu items
- Add/remove items from cart
- Order management
- Routing with React Router
- State management using Redux Toolkit
- Responsive design

## Technologies Used

- React.js
- Redux Toolkit
- React Router
- JavaScript
- CSS
- Vite

## Installation

```bash
npm install
npm run dev
```
